ADMIN CONTENT SYSTEM REBUILD — GENERATIVE AI + MEDIA BOXES

Current admin is still too technical and fragmented.

We need to redesign it around CONTENT CREATION PER GAME.

Goal:
Every game must have ONE simple page where the Super Admin can:
- create new themes
- generate content with AI
- manage active/inactive themes
- manage static media assets tied to that game

This must be visual, fast, and obvious.

==================================================
PART 1 — GENERATIVE AI CONTENT FLOW (ALL GAMES)
==================================================

When entering a game from Admin
(example Adult Only / SaraMusica / Quiz / Coppie / Parola alle Spalle...)

Top section always visible:

-----------------------------------
+ NUOVO TEMA
[ nome tema input ]
[ difficoltà ▼ easy / medium / hard ]
[ GENERA ]
-----------------------------------

GENERA triggers AI generation.

AI can generate:
- text
- prompts
- placeholder images
- question sets
- challenge sets
- phrase sets

AI DOES NOT generate audio.

Audio comes from:
- YouTube
OR
- static DB media library

==================================================
PART 2 — GENERATED THEMES LIST
==================================================

Below generator:

list of themes already created for that game.

Each theme card shows:

Theme name
Difficulty
Created date
Items count

toggles:
[ Home ON/OFF ]
[ Live ON/OFF ]

Meaning:
HOME ON → appears in Home mode
LIVE ON → appears in Live mode

Actions:
Edit
Duplicate
Delete
Preview

This structure must exist for EVERY GAME.

PART 2 — WORD BACK PLAYER ROTATION EVERY 3 WORDS
==================================================

Current:
The same INDOVINO/SUGGERITORE can continue too long.

Required:
Every 3 words, the game must change players.

Logic:
- One pair plays 3 words.
- After 3 words, return to booking phase.
- New players can book as:
  - INDOVINO
  - SUGGERITORE
- Booking window lasts 10 seconds.

If both roles are booked within 10 seconds:
- continue with those players.

If 10 seconds expire and roles are incomplete:
- show all connected player names on TV
- randomly select 2 connected players
- assign:
  random player 1 = INDOVINO
  random player 2 = SUGGERITORE
- if only one connected player exists, show error:
  “Servono almeno 2 giocatori”
- if current pair is still connected, they can be selected again only if there are fewer than 4 players connected; otherwise prefer players who were not in previous pair.

Server state:
Track in roundPayload or session gameConfig:
{
  wordBackPairRoundCount: number,
  wordBackCurrentPair: {
    guesserId,
    suggesterId
  },
  wordBackPairHistory: []
}

After each word/round:
- increment wordBackPairRoundCount
- if count < 3:
  continue next word with same pair
- if count === 3:
  reset count to 0
  clear bookedPlayers
  enter booking phase for WordBack
  start 10-second booking window

Booking phase UI:
TV:
- show:
  “Cambio giocatori”
  “Prenotatevi entro 10 secondi”
  countdown 10s
  role slots:
    INDOVINO
    SUGGERITORE
- after timeout:
  show random selection animation:
  “Scelta casuale giocatori…”
  then selected pair

Phones:
- show two role buttons:
  “MI PRENOTO COME INDOVINO”
  “MI PRENOTO COME SUGGERITORE”
- if role occupied, disable that role.
- after timeout/random pick, selected players see their roles.

Events/API:
Add/patch:
- home:wordback_pair_rotation_start
- home:wordback_random_pair_selected
or implement via existing home:state if simpler.

Important:
- Do not break taboo alarm.
- Do not break speech answer validation.
- Do not break +100/+100 scoring.
- Do not show word to indovino.
- TV still shows word + taboo words.

Output:

E. random fallback selection

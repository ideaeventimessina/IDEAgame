PRIORITY 4 — GIOCO DELLE COPPIE PHONE/TV SEPARATION
==================================================

Problems:
- Phone players behave like connected TV screens.
- If one player touches a card, all phones reveal the same card.
- Cards are revealed on all 4 screens.
- Players cannot independently click if another player already touched.
- TV card images too large/overflow, bottom row hidden under bar.

Required:
Separate TV board state from player phone controller.

TV:
- shows shared board.
- large images, but must fit viewport.
- bottom row must never be hidden by footer/bar.
- use safe container height and responsive grid.

Phones:

  phones show controls/actions, not full shared card board.
  TV owns card reveal state.

Patch:

- visibility preview 10s may be broadcast to phones only as preview, not as permanent card reveal.
- ensure preview state does not conflict with active card selection.
- fix TV grid:
  max-height: calc(100dvh - footerHeight)
  overflow hidden/scale grid
  card size clamp()
  no bottom overlap.

Goal:
When player taps “ENTRA NELLA STANZA”, run a device preflight so games start faster and with fewer interruptions.

On the same user gesture as Enter Room:
1. unlock audio context
2. request motion/orientation permission if available
3. request microphone permission
4. detect SpeechRecognition support
5. save capability state
6. continue joining the room regardless of granted/denied

Important:
Do NOT block room entry.
Do NOT force permissions.
Do NOT show scary technical errors.
